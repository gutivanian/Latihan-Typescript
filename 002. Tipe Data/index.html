Tipe Data

Typescript mensupport beberapa tipe data primitif:

  • boolean (true atau false)
  • number (angka dan float)
  • string
  • bigint (nilai angka dan float yang sangat besar)
  • symbol (biasanya untuk identifier global)

Menambahkan Tipe

Ketika membuat sebuah variabel, dalam typescript bisa menggunakan cara eksplisit dan implisit

Tipe Eksplisit

Menuliskan tipe secara eksplisit dilakukan saat pendefinisian variabel. Penambahan tipe secara eksplisit lebih mudah dibaca.

let firstName : string = "Nama"

Tipe Implisit

Typescript akan menebak tipe berdasarkan nilai yang diinput. Proses pendefinisian variabel ini sama saja dengan javascript biasa. Proses typescript untuk menebak tipe dari sebuah variabel nilai disebut sebagai "infer". Proses penambahan tipe secara implisit dengan infer lebih singkat, cepat dan biasa digunakan ketika melakukan development dan testing.
