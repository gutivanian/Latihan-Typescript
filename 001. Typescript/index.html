Typescript

Typescript adalah Javascript dengan tambahan sintaks untuk "type". Typescript adalah "syntactic superset" dari javascript dengan tambahan "static typing". Ini sama saja bahwa Typescript menambahkan sintaks diatas javascript, mengijinkan developer untuk menambahkan "tipe". Typescript menjadi "Syntactic superset" berarti bahwa dia berbagi sintaks yang sama dengan javascript, namun menambahkan sesuatu didalamnya.

Mengapa Perlu TypeScript?

Javascript adalah bahasa dengan tipe yang bebas. Ini berarti cukup sulit untuk memahami tipe data apa yang digunakan di sepanjang javascript. Pada Javascript, parameter fungsi dan variabel tidak memiliki informasi apapun, sehingga developer perlu untuk melihat dokumentasinya atau menebak-nebak dalam penggunaannya.

Typescript mengijinkan untuk menspesifikasikan tipe data yang digunakan di sepanjang kode dan dapat melemparkan error ketika tipe tidak sesuai. Sebagai contoh, typescript akan melemparkan error ketika memasukkan string ke fungsi yang harusnya butuh angka. Javascript akan tetap mengijinkan. Typescript akan mengecek bahwa tipe data sesuai sebelum menjalankan kode, bukan saat menjalankan kode.

Bagaimana Cara Menggunakan Typescript?

Salah satu cara menggunakan Typescript adalah menggunakan compiler Typescript official yang akan mengubah kode Typescript ke Javascript. Kita juga bisa menginstall compiler di komputer lokal. Code editor seperti Visual Studio Code mensupport Typescript dan dapat menampilkan error pada kode yang kita tulis.
