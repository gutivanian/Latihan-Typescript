Typescript Enums

Enum adalah class khusus yang merepresentasikan kelompok konstanta (variabel yang tidak berubah). Enum memiliki dua variasi yaitu string dan numerik.

Enum Numerik - Default

Secara default, enum akan menginisiasi nilai awalnya ke "0" dan akan menambah 1 untuk setiap nilai tambahan. 

        enum CardinalDirections {
          North,
          East,
          South,
          West
        }
        let currentDirection = CardinalDirections.North;
        // logs 0
        console.log(currentDirection);
        // throws error as 'North' is not a valid enum
        currentDirection = 'North'; // Error: "North" is not assignable to type 'CardinalDirections'.

Enum Numerik - Dengan Inisialisasi

kita bisa menyetting nilai awal dari enum dan kemudian akan secara otomatis bertambah nilainya ketika ditambah item di dalam enum 

        enum CardinalDirections {
          North,
          East,
          South,
          West
        }
        let currentDirection = CardinalDirections.North;
        // logs 0
        console.log(currentDirection);
        // throws error as 'North' is not a valid enum
        currentDirection = 'North'; // Error: "North" is not assignable to type 'CardinalDirections'.

Enum Numerik - Inisialisasi Penuh

Kita bisa menambahkan nilai unik untuk setiap komponen pada enum 

        enum StatusCodes {
          NotFound = 404,
          Success = 200,
          Accepted = 202,
          BadRequest = 400
        }
        // logs 404
        console.log(StatusCodes.NotFound);
        // logs 200
        console.log(StatusCodes.Success);
