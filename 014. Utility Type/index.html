Typescript Utility Type

Utility types dalam TypeScript adalah jenis-jenis yang disediakan oleh TypeScript untuk memudahkan manipulasi dan transformasi tipe yang ada. Utility types ini membantu dalam berbagai operasi seperti membuat tipe baru berdasarkan tipe yang sudah ada, menggabungkan tipe, atau bahkan membuat tipe yang lebih ketat atau lebih longgar.

Partial

"Partial" mengubah semua properti pada sebuah objek menjadi opsional. 

        interface Point{
            x:number;
            y:number
        }
        
        let pointPart:Partial<Point> = {} 
        // partial penjadikan x dan y opsional
        pointPart.x = 10;

Required

Required mengubah semua properti pada objek bersifat wajib. 

        interface Car{
            make: string;
            model: string;
            mileage?: Number;
        }
        
        let myCar: Required<Car> = {
            make: 'Ford',
            model: 'Focus',
            mileage: 1000
        }

Record

Record adalah jalan pintas untuk mendefinisikan sebuah tipe objek dengan tipe key khusus dan tipe value. 

        const nameAgeMap: Record<string, number> = {
            'Alice': 25,
              'Bob': 22
          }

Omit

Omit mengubah key dari sebuah tipe object. 

        interface Person {
            name:string;
            age: number;
            location?: string
        }
        
        const bob:Omit<Person, 'age'|'location'> = {
            name: 'Bob'
            // omit telah menghilangkan age dan location
        }

Pick

Pick membuang semua key kecuali key tertentu yang dipilih. 

        interface Person{
            name:string;
            age: number;
            location?: string;
        }
        
        const bob: Pick<Person, 'name'> = {
            name: "Bob"
            // pick hanya mengambil name saja
        }

Exclude

Exclude membuang tipe dari uniwon. 

        type Primitive = string | number | boolean;
        const value: Exclude<Primitive,string>  = true;
        // string tidak lagi dapat digunakan karena telah dibuang dary type

ReturnType

ReturnType mengekstrak tipe return dari sebuah fungsi. 

        type PointGenerator = () => {
            x:number;
            y:number
        }
        
        const point:ReturnType<PointGenerator> = {
            x:10,
            y:15
        }

Parameters

Parameters mengekstrak tipe parameter dari sebuah fungsi sebagai sebuah array 

        type PointPrinter = (p: {x: number; y: number}) =>
    void 

        const point: Parameters<PointPrinter>[0]={
            x:10,
            y:20
        }

Readonly

Readonly digunakan untuk membuat tipe baru dimana semua propertinya bersifat read-only. 

        interface Person{
            name:string;
            age:number
        }
        
        const person: Readonly<Person> = {
            name: "John",
            age: 30
        }
        
        // person.name = "asu" 
        // akan error
