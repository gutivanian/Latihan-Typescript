Fungsi Typescript

Typescript memiliki tipe khusus untuk memberikan tipe pada parameter fungsi dan nilai return.

Tipe Return

Tipe dari nilai yang direturn oleh fungsi dapat didefinisikan secara eksplisit. 

        // the `: number` here specifies that this function returns a number
        function getTime(): number {
        return new Date().getTime();
        }

Tipe Return Void

Tipe "void" dapat digunakan untuk fungsi yang tidak mengembalikan nilai apapun 

        function printHello(): void {
            console.log('Hello!');
        }

Parameter

Parameter fungsi juga dapat diberi tipe dengan sintaks yang sama dengan saat deklarasi variabel. Jika tidak ada tipe parameter yang didefinisikan, Typescript akan memberikan tipe default ke "any". 

        function multiply(a: number, b: number) {
            return a * b;
          }

Parameter Opsional

Secara default, Typescript akan mengasumsikan bahwa semua parameter diperlukan, namun kita bisa membuatnya opsional. 

        // the `?` operator here marks parameter `c` as optional
        function add(a: number, b: number, c?: number) {
        return a + b + (c || 0);
        }

Parameter Default

Untuk parameter dengan nilai default, nilai default didefinisikan setelah deklarasi tipe. 

        function pow(value: number, exponent: number = 10) {
            return value ** exponent;
        }

Parameter Dinamai

Menuliskan parameter dengan penamaan sama saja dengan menulis tipe parameter secara normal. Dalam contoh ini, divide adalah sebuah fungsi yang menerima satu parameter, yaitu sebuah objek yang memiliki dua properti: dividend dan divisor. Kedua properti ini bertipe number.
