Tipe Khusus TypeScript

Typescript memiliki tipe khusus yang mungkin tidak terkait engan tipe data primirif

Tipe any

any adalah tipe yang tidak mengaktifkan pengecekan tipe dan secara efektif mengijinkan semua tipe untuk digunakan. Contoh dibawah tidak menggunakan tipe any dan akan menghasilkan error 

        let u = true;
        u = "string"; // Error: Type 'string' is not assignable to type 'boolean'.
        Math.round(u); // Error: Argument of type 'boolean' is not assignable to parameter of type 'number'.

Dan kita bisa menggunakan tipe any untuk mengijinkan semua tipe data digunakan 

        let v: any = true;
        v = "string"; // no error as it can be "any" type
        Math.round(v); // no error as it can be "any" type
