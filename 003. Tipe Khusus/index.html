Tipe Khusus TypeScript

Typescript memiliki tipe khusus yang mungkin tidak terkait engan tipe data primirif

Tipe any

any adalah tipe yang tidak mengaktifkan pengecekan tipe dan secara efektif mengijinkan semua tipe untuk digunakan. Contoh dibawah tidak menggunakan tipe any dan akan menghasilkan error 

        let u = true;
        u = "string"; // Error: Type 'string' is not assignable to type 'boolean'.
        Math.round(u); // Error: Argument of type 'boolean' is not assignable to parameter of type 'number'.

Dan kita bisa menggunakan tipe any untuk mengijinkan semua tipe data digunakan 

        let v: any = true;
        v = "string"; // no error as it can be "any" type
        Math.round(v); // no error as it can be "any" type

Tipe Unknown

unknown adalah tipe khusus yang mungkin tidak terkait engan tipe data primitif. Tipe unknown akan memberikan tipe data "any" dan secara efektif mengijinkan semua tipe data untuk digunakan. unknown mirip dengan any, namun lebih aman ketimbang any. Typescript akan melarang tipe unknown untuk digunakan. Untuk memanggil varibel any, kita perlu menambahkan/mendefinisikan tipe data dulu dengan "casting". Casting adalah ketika kita menggunakan kata kunci "as" untuk memanggil properti/variabel sesuai dengan tipe casting. 

        let w: unknown = 1;
        w = "string"; // no error
        w = {
          runANonExistentMethod: () => {
            console.log("I think therefore I am");
          }
        } as { runANonExistentMethod: () => void}
        // How can we avoid the error for the code commented out below when we don't know the type?
        // w.runANonExistentMethod(); // Error: Object is of type 'unknown'.
        if(typeof w === 'object' && w !== null) {
          (w as { runANonExistentMethod: Function }).runANonExistentMethod();
        }
        // Although we have to cast multiple times we can do a check in the if to secure our type and have a safer casting

Tipe never

never adalah tipe khusus yang digunakan untuk menandai bahwa suatu fungsi tidak akan mengembalikan nilai. Contoh dibawah adalah fungsi yang tidak akan mengembalikan nilai. 

        let x: never = true; // Error: Type 'boolean' is not assignable to type 'never'.

Tipe Undefined dan Null

"undefined" dan "null" adalah tipe data yang berkaitan ke tipe primitif JS "undefined" dan "null" 

        let y: undefined = undefined;
        let z: null = null;
