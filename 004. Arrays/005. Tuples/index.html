Tuple

Tuple adalah tipe data yang berisikan kumpulan data. Typescript memiliki sintaks khusus untuk penulisan tuple. Tuple mirip dengan array, namun perlu didefinisikan masing-masing tipe anggotanya. Array dalam tuple dapat memiliki bermacam-macam tipe data. 

        // define our tuple
        let ourTuple: [number, boolean, string];
        
        // initialize correctly
        ourTuple = [5, false, 'Coding God was here'];

Seperti yang terlihat bahwa tipe data dalam tuple adalah number, boolean, dan string. Namun, apa yang akan terjadi jika kita mengisi data berbeda urutan? tentu akan error. 

        // define our tuple
        let ourTuple: [number, boolean, string];
        
        // initialized incorrectly which throws an error
        ourTuple = [false, 'Coding God was mistaken', 5];

Tuple Readonly

Kita bisa membuat tuple bersifat read-only. Tuple hanya perlu didefinisikan tipenya untuk nilai awal. Ini adalah contoh untuk tuple yang tidak read-only 

        // define our tuple
        let ourTuple: [number, boolean, string];
        // initialize correctly
        ourTuple = [5, false, 'Coding God was here'];
        // We have no type safety in our tuple for indexes 3+
        ourTuple.push('Something new and wrong');
        console.log(ourTuple);
