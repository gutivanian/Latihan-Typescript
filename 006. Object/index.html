Typescript Tipe Object

Typescript memiliki sintaks khusus untuk meniliskan tipe object. 

        const car: { type: string, model: string, year: number } = {
          type: "Toyota",
          model: "Corolla",
          year: 2009
        };

Inferensi Tipe

Typescript dapat infer/menebak tipe dari properti berdasarkan nilainya. 

        const car = {
          type: "Toyota",
        };
        car.type = "Ford"; // no error
        car.type = 2; // Error: Type 'number' is not assignable to type 'string'.

Properti Opsional

Properti opsional adalah properti yang tidak perlu didefinisikan di dalam object. Jika tanpa properti opsional, maka properti wajib didefinisikan didalam objek, jika tidak akan memberikan error
