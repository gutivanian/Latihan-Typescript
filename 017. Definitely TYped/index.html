DefinitelyTyped

DefinitelyTyped adalah sebuah repositori yang berisi deklarasi tipe TypeScript untuk berbagai pustaka JavaScript pihak ketiga. Ini memungkinkan pengembang untuk menggunakan pustaka JavaScript yang tidak ditulis dalam TypeScript dengan keamanan tipe yang disediakan oleh TypeScript.

Mengapa DefinitelyTyped Penting?

  • Keamanan Tipe: Menambahkan tipe untuk pustaka JavaScript memungkinkan pengembang untuk menangkap kesalahan tipe pada waktu kompilasi daripada waktu runtime.
  • Pengalaman Pengembangan yang Lebih Baik: Editor teks dan IDE dapat memberikan auto-completion, linting, dan dokumentasi inline saat bekerja dengan pustaka yang memiliki deklarasi tipe.
  • Interoperabilitas: Memungkinkan pustaka JavaScript yang ada untuk digunakan dalam proyek TypeScript tanpa harus menulis ulang pustaka tersebut dalam TypeScript.

Menggunakan DefinitelyTyped

Pustaka-pustaka dalam DefinitelyTyped tersedia melalui paket npm dengan nama @types/<nama-paket>. Berikut adalah langkah-langkah untuk menggunakannya:

Instalasi

Untuk menginstal tipe untuk pustaka tertentu, Anda menggunakan npm atau yarn. Misalnya, jika Anda ingin menambahkan tipe untuk lodash, Anda bisa menjalankan:

npm install --save-dev @types/lodash

atau dengan yarn:

yarn add @types/lodash --dev

Penggunaan

Setelah tipe terinstal, TypeScript akan secara otomatis mengenali tipe tersebut. Anda bisa langsung mengimpor dan menggunakan pustaka tersebut dalam kode TypeScript Anda:

import _ from 'lodash';

let array = [1, 2, 3, 4];
let reversedArray = _.reverse(array);

console.log(reversedArray); // Output: [4, 3, 2, 1]

Struktur DefinitelyTyped

DefinitelyTyped adalah sebuah repositori besar di GitHub yang diorganisasikan sebagai berikut:

  • Repositori: DefinitelyTyped GitHub
  • Struktur Direktori: Setiap pustaka memiliki direktori sendiri di dalam direktori types. Misalnya, tipe untuk lodash terletak di types/lodash.

Contoh Isi Paket Tipe

Setiap direktori pustaka di DefinitelyTyped biasanya berisi beberapa file seperti:

  • Index.d.ts: File utama yang berisi deklarasi tipe.
  • tsconfig.json: Konfigurasi TypeScript untuk memastikan tipe tersebut benar.
  • Tests: Berkas pengujian untuk memastikan tipe bekerja dengan benar.

Contoh dari isi types/lodash/index.d.ts mungkin seperti ini:

// Type definitions for Lo-Dash 4.14
// Project: http://lodash.com/
// Definitions by: Brian Zengel <https://github.com/bczengel>
// Definitions: https://github.com/DefinitelyTyped/DefinitelyTyped

export = _;
export as namespace _;

declare const _: _.LoDashStatic;

declare namespace _ {
interface LoDashStatic {
    reverse<T>(array: T[]): T[];
    // ... banyak fungsi lainnya
}
}

Menyumbang ke DefinitelyTyped

Jika Anda menemukan pustaka yang tidak memiliki tipe atau tipe yang ada perlu perbaikan, Anda bisa menyumbang ke DefinitelyTyped:

  1. Fork Repositori: Fork repositori DefinitelyTyped di GitHub.
  2. Buat Cabang Baru: Buat cabang baru untuk perubahan Anda.
  3. Tambahkan atau Perbarui Tipe: Tambahkan atau perbarui deklarasi tipe.
  4. Tambah Pengujian: Tambahkan pengujian untuk memastikan tipe Anda bekerja dengan benar.
  5. Buat Pull Request: Kirim pull request ke repositori utama DefinitelyTyped.

Kesimpulan

DefinitelyTyped adalah sumber daya penting bagi pengembang TypeScript yang bekerja dengan pustaka JavaScript pihak ketiga. Dengan menyediakan deklarasi tipe, DefinitelyTyped memungkinkan penggunaan pustaka JavaScript dengan keamanan tipe, memberikan pengalaman pengembangan yang lebih baik, dan membantu dalam penangkapan kesalahan pada waktu kompilasi. Menggunakan DefinitelyTyped adalah praktik yang baik untuk memastikan kode TypeScript Anda tetap kuat dan mudah dipelihara.
